100-Day Learning Roadmap

Follow each phase step by step and build a strong ML portfolio.

Days 1–10: Foundation

Python, NumPy, pandas, datasets, and preprocessing.

Days 11–25: Regression

Linear regression, multiple regression, polynomial regression, and metrics.

Days 26–45: Classification

Logistic regression, KNN, SVM, Naive Bayes, and confusion matrix.

Days 46–65: Tree Models

Decision trees, random forests, feature importance, and overfitting control.

Days 66–80: Unsupervised ML

K-means clustering, PCA, pattern discovery, and segmentation.

Days 81–100: Portfolio Projects

End-to-end projects, README writing, reports, and GitHub publishing.
