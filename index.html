Roadmap

100-Day Machine Learning Learning Plan

Follow this roadmap consistently to build practical ML skills and publish your progress on GitHub.

Days 1–10: Foundations

Python basics, NumPy, pandas, data cleaning, missing values, encoding, scaling, and train-test split.

Days 11–25: Regression

Simple linear regression, multiple regression, polynomial regression, evaluation metrics, and prediction projects.

Days 26–45: Classification

Logistic regression, KNN, SVM, Naive Bayes, confusion matrix, precision, recall, and F1-score.

Days 46–65: Tree-Based Models

Decision trees, random forests, feature importance, overfitting, pruning, and model comparison.

Days 66–80: Unsupervised Learning

K-means clustering, dimensionality reduction, PCA, data patterns, and customer segmentation projects.

Days 81–100: Portfolio Projects

Build, document, and publish complete ML projects with clean README files, datasets, notebooks, and results.

Modules

What You Will Learn

Each module is designed to move from concept to implementation and then to portfolio output.

Python & Data Tools

Python, Jupyter, NumPy, pandas, matplotlib, CSV files, and clean coding habits.

Data Preprocessing

Handle missing values, categorical data, scaling, splitting, and preparing datasets for ML.

Supervised Learning

Regression, classification, prediction, model training, and evaluation metrics.

Unsupervised Learning

Clustering, PCA, pattern discovery, and exploratory data analysis.

Model Evaluation

Accuracy, precision, recall, F1-score, RMSE, MAE, cross-validation, and error analysis.

GitHub Portfolio

Write professional README files, publish notebooks, add screenshots, and explain results clearly.
