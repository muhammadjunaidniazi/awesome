# Agri Aqua Advisor

A modern Computer Science subject project for smart farming and irrigation planning. This is an alternative version of a farming application with a completely different UI/UX style and a lightweight offline-first design.

## Live/Local Features

- Smart irrigation recommendation by crop, soil, moisture, temperature, and field area
- Moisture comparison chart using HTML Canvas
- Saved field records using browser LocalStorage
- Responsive mobile-friendly UI
- No database required
- No paid API required
- Works by opening `index.html` in a browser

## Tech Stack

- HTML5
- CSS3
- JavaScript
- Canvas API
- LocalStorage

## How to Run

1. Download or clone this project.
2. Open `index.html` in any modern browser.
3. Enter field details and generate the irrigation plan.

## Project Use

This project can be submitted as a CS subject project or uploaded to GitHub as a portfolio project.
