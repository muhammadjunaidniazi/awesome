# ML Mastery Roadmap

A professional beginner-friendly machine learning learning path designed as a clean alternative to a simple 100-day machine learning code repository. This repository is organized as a practical course-style roadmap for students, beginners, and developers who want to learn machine learning step by step and build portfolio-ready projects.

## Live Website

Website / GitHub Pages link: https://muhammadjunaidniazi.github.io/awesome/

## Course Goal

The goal of this repository is to help learners understand machine learning from the basics and practice it through Python, datasets, algorithms, model evaluation, and real projects.

## What You Will Learn

- Python for machine learning
- NumPy, pandas, and data handling
- Data preprocessing and feature preparation
- Regression models
- Classification models
- Decision trees and random forests
- Clustering and unsupervised learning
- Model evaluation and comparison
- Portfolio-ready machine learning projects
- GitHub documentation and project presentation

## 100-Day Learning Roadmap

| Phase | Days | Focus |
|---|---:|---|
| Foundation | 1–10 | Python, NumPy, pandas, preprocessing, datasets |
| Regression | 11–25 | Linear regression, multiple regression, polynomial regression |
| Classification | 26–45 | Logistic regression, KNN, SVM, Naive Bayes |
| Tree Models | 46–65 | Decision tree, random forest, feature importance |
| Unsupervised ML | 66–80 | K-means clustering, PCA, pattern discovery |
| Portfolio Projects | 81–100 | End-to-end ML projects, README, reports, GitHub publishing |

## Portfolio Project Ideas

- Student score prediction
- Startup profit prediction
- Customer purchase prediction
- Social network ads classification
- Customer segmentation
- End-to-end machine learning report
